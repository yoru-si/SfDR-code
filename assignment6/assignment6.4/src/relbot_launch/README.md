Launch file space:


Contains `relbot_turtlesim_launch.py`  which runs cam2image, showimage, relbot_simulator, relbot2turtlesim, turtlesim_node, relbot_sequence_controller nodes.
Contains `relbot_test_launch.py` which runs cam2image, relbot_adapter, relbot_simulator.

#### Run:
Build and source the workspace as mentioned in the home directory README.txt and type the following:
* `ros2 launch relbot_launch relbot_turtlesim_launch.py`

* `ros2 launch relbot_launch relbot_test_launch.py`
