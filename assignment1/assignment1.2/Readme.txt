Assignment 1.2 - BMI Calculator
________________________________

1. Build Command:
g++ src/*.cpp -iquote include -o assignment1.2.exe

2. Run Command
Windows: .\assignment1.2.exe

3. What it does:
- Asks the user(you) to input desired body weight(in kgs) and height(in m)
- Computes BMI
- Prints BMI value, BMI category, and the NIH/DHHS BMI information table


Locate files:
- Functions src/BMI.cpp
- Function Declarations include/BMI.h
- Main file with user interactions src/main.cpp
